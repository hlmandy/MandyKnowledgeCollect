#### 相关系数

cc_corr = cc_pd.corr(method='spearman')  #相关系数矩阵  

method中，有三种相关系数的计算方式，包括 —— 'pearson', 'kendall', 'spearman'，常用的是线性相关pearson。 

但pearson对于异常值很敏感。如果有异常值，可以用spearman

```python
cc_corr = cc_pd.corr(method='pearson') #相关系数矩阵
```
