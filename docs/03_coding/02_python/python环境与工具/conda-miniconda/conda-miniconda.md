## 虚拟环境

在 powershell中输入

初始化

```
conda init

```

创建名为[openmmlab](https://so.csdn.net/so/search?q=openmmlab\&spm=1001.2101.3001.7020 "openmmlab")的python3.9虚拟环境

```
conda create -n openmmlab python=3.9

```

激活虚拟环境

```
conda activate openmmlab

```

退出虚拟环境

```
conda deactivate

```
